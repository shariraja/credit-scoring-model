# Credit Scoring Model 💳

A machine learning project to predict credit risk (Good/Bad) using the German Credit Dataset.

## Project Overview
- **Task**: Binary Classification (Good / Bad Credit Risk)
- **Dataset**: German Credit Data (954 records)
- **Models Used**: Logistic Regression, Random Forest
- **Best Model**: Logistic Regression (Recall: 0.78, ROC-AUC: 0.75)

## Tech Stack
- Python, Scikit-learn, Pandas, NumPy
- Streamlit (Web App)
- Jupyter Notebook

## Features Used
- Age, Sex, Job, Housing
- Saving Accounts, Checking Account
- Credit Amount, Duration, Purpose

## How to Run
pip install -r requirements.txt
streamlit run app.py
